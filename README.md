# LLM Evaluation Dashboard

This project is a Streamlit-based dashboard used to evaluate LLM responses.

## Features
- Compare LLM outputs
- Display results using charts
- Simple UI using Streamlit

## Tech Stack
- Python
- Streamlit
- Matplotlib

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   streamlit run app.py
